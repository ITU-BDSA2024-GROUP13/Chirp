@startuml
autonumber

actor User as UU
boundary Browser as B
control WebServer as WS
entity TimeLine as PT
control CheepService as CS
control CheepRepository as CR
database Database as DB

' Step 1: User searches
UU-> B: Type into Search Input
B-> B: Trigger input event
B-> WS: POST {SearchString:input}
WS-> PT: OnPostSearch(SearchRequest)

' Step 2: Service searches
PT-> PT: Extract SearchString
PT-> CS: Find Authors
CS->CR: FindAuthors(searchRequest.SearchString)
CR-> DB : Query matching users

' Step 3: List of authors returned
DB --> CR: Return Author list
CR --> CS: Return Author<DTO>s
CS --> PT: Return JSON { list: [...] }
PT --> WS : Return JSON Response
WS --> B: Return JSON Response

' Step 4: Searches displayed
B-> B: Call showResults()
B-> B: Create DOM Elements
B-> UU: Display Search Results
@enduml
