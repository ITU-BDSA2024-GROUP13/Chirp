@startuml
actor User
participant "Browser (JavaScript)" as Browser
participant "Web Server" as Server
participant "Timeline" as Handler
participant "CheepService" as Service
database "Repository/Database" as DB

User -> Browser : Type into Search Input
Browser -> Browser : Trigger input event
Browser -> Server : POST /?handler=Search\n{ SearchString: input }
Server -> Handler : OnPostSearch()
Handler -> Handler : Extract SearchString
Handler -> Service : Call SearchService(SearchString)
Service -> DB : Query matching users
DB --> Service : Return user list
Service --> Handler : Return user DTOs
Handler --> Server : Return JSON Response
Server --> Browser : JSON { list: [...] }
Browser -> Browser : Call showResults()
Browser -> Browser : Create DOM Elements
Browser -> User : Display Search Results
@enduml
